Pour compiler un fichier, insérez dans un invite de commande :

Compiler.py CHEMIN_DU_FICHIER

Le résultat de la compilation se trouve dans le dossier "compile" qui contient directement le framework Reveal.

Vous pouvez directement tester les différents exemples déjà présents dans ce dossier :

Compiler.py examples/example-01.txt
Compiler.py examples/example-02.txt
Compiler.py examples/example-03.txt
Compiler.py examples/presentation.txt
